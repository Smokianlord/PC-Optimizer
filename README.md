PC Optimizer v2 is a simple Windows maintenance app built with Python.  
It gives users one clean control panel to run common optimization and repair tasks without manually opening multiple batch files.

## Features

- Clean temporary Windows files
- Reset internet/network settings
- Repair Windows system files
- Close unnecessary background apps
- Check system requirements
- Run full optimization sequence
- Customer-friendly interface
- 3D-style buttons
- Admin restart option
- Custom app icon

## Tools Included

### Clean Temporary Files
Removes unnecessary temporary files from Windows temp folders, user temp folders, and prefetch.

### Reset Internet Connection
Runs common network reset commands to fix DNS, IP, Winsock, and connection issues.

### Repair Windows System
Uses Windows repair tools such as DISM, SFC, and CHKDSK.

### Close Background Apps
Closes common unnecessary background apps to free CPU and RAM.

### Check Requirements
Checks Windows compatibility, administrator status, and required commands.

### Run Full Optimization
Runs the main optimization tasks in sequence.
